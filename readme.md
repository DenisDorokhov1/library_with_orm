### Library Management API
Простое REST API на Flask для управления библиотекой. Система позволяет отслеживать наличие книг, фиксировать выдачу литературы студентам и анализировать читательскую активность.

### Основные возможности
* Учет фонда: Хранение данных об авторах и книгах, с учетом количества экземпляров.

* Выдача книг: Регистрация выдачи книг студентам с автоматическим уменьшением остатка в библиотеке.

* Умные рекомендации: Эндпоинт для поиска непрочитанных книг тех авторов, которых студент уже выбирал ранее.

### Аналитика:

* Прогноз среднего количества взятых книг за текущий месяц.

* Поиск самой популярной книги среди отличников (средний балл > 4.0).

* Топ самых активных читателей текущего года.

* Импорт данных: Массовая загрузка студентов через CSV-файлы.

### Структура проекта
Для удобства поддержки код разделен на логические модули:

* `data.py` — Описание моделей БД (SQLAlchemy), настройка связей (relationship, association_proxy) и логика инициализации базы.

* `routes.py` — Определение API-эндпоинтов, обработка запросов и бизнес-логика.

* `library.db` — Файл базы данных SQLite (создается автоматически при первом запуске `data.py`).

Полная версия объединена в один код `api.py`

## Установка (Linux)

1. Клонируем репозиторий 


2. Создаем виртуального окружения

```python3 -m venv venv```

3. Активация виртуального окружения

```source venv/bin/activate```

4. Установка зависимостей

```pip install -r requirements.txt```

5. Запуск Docker-контейнера с приложением

```docker compose up```
